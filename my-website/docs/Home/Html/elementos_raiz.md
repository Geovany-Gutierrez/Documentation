**Conceito e Uso:**
Os elementos raiz, incluindo `<html>`, formam a estrutura básica de um documento HTML, encapsulando todo o conteúdo da página.

**Exemplo de código:**
```html
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <!-- Conteúdo da página aqui -->
</body>
</html>
```
